Описание программы
Программа ESP_MESH_Creator предназначена для быстрого создания mesh сети, путем непосредственной загрузки готового кода с необходимыми значениями переменных на микроконтроллер.  
Сама программа написана на языке программирования C# в среде разработки Visual Studio 2019 с использованием рабочей среды .NET Framework 4.7.2.

Инструкция по использованию программного обеспечения

Для начала работы с программой необходимо запустить ESP_Mesh_creator.exe.
При запуске программы может появиться уведомление об отсутствии доступных COM-портов, это связано с тем, что связь с платой на которую будет загружаться код отсутствует. В случае наличия соединения уведомление не всплывет.
 
В верхнем левом поле указывается имя сети, в которой будут взаимодействовать узлы между собой. Само поле позволяет вводить только латинские буквы и знак «_». Количество символов должно быть больше 7 и меньше 40.
 
Следующим полем для запыления идет поле ввода пароля, которое имеет такие же ограничения, что и предыдущее. 
Поле «IP Port» предназначено для ввода порта для настройки и корректной работы сети. 

Поле «Номер узла» позволяет задать контрейлеру, на который будет загружаться программа, свой собственный ID в создаваемой сети.

«COM Port» позволяет выбрать порт из доступных, через который будет загружаться код непосредственно на контроллер.

После ввода всех необходимых переменных и выбора порта для загрузки кода, программа предоставляет возможность сохранить код на компьютер и непосредственно загрузить прошивку на плату, присоединенную к указанному COM-потру
Также у программы есть возможность не только загружать встроенный код но и выбрать свой код. Для этого необходимо поставить галочку в поле «Воспользоваться своим кодом» и нажать на клавишу «Выбрать свой код». Важно, для корректной работы программы при использовании своего кода необходимо придерживаться названия переменных используемой программы. Что узнать о доступных переменных, с которыми работает программа, необходимо нажать на клавишу «INFO», тогда появиться окно с информацией о доступных переменных.
 

Для того, чтобы изменить встроенный код на другой необходимо изменить его в файле work_code_arduino.txt.
